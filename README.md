## System Requirements
    - PHP >= 7.2.5
    - MySql 5.7

# Configuration
    - cp .env.example .env

## Run the database migrations

    - php artisan migrate

## Coding convention

    - Check: `php vendor/bin/phpcs --standard=phpcs-laravel app/`

    - Fix: `php vendor/bin/phpcbf --standard=phpcs-laravel app/`
